Figure 1:
Misalkan kerucut terletak di bidang xy dengan alas berjejari R dan tinggi T, sumbu kerucut berimpit dengan sumbu z.
Ambil elemen massa berupa cakram dengan jejari r yang terletak pada posisi z dan tebalnya dz
Dengan jejari R dan tinggi T, maka penampang vertikal kerucut berbentuk dua segitiga siku-siku dengan sisi siku-siku R dan T dan sudut alas adalah θ = arctan (T/R). Maka elemen massa cakram yang terletak pada ketinggian z akan mempunyai jejari
r/(T-z)=R/T --> r = (R/T)*(T-z)
jika tebal elemen massa tersebut adalah dz, maka volume elemen adalah
dV=π*r^2*dz=π*(T-z)^2(R/T)^2dz
dan massa elemen adalah
dM=ρdV=ρ*π*(T-z)^2(R/T)^2dz
(%i5) T : 2 $ /*tinggi kerucut*/
R : 1 $ /*jejari alas kerucut*/
r ( z ) : = ( R / T ) * ( T - z ) $ /*jejari cakram pada posisi z*/
dVol ( z ) : = %pi * r ( z ) * r ( z ) $ /*volume cakram*/
dM ( z ) : = %rho * dVol ( z ) ;

\[\operatorname{ }\operatorname{dM}(z)\operatorname{:=}\rho \operatorname{dVol}(z)\]

Dengan demikian volume kerucut tersebut adalah
(%i6) V_kerucut : integrate ( dVol ( z ) , z , 0 , T ) ;

\[\operatorname{ }\frac{2 \ensuremath{\pi} }{3}\]

Massa kerucut
(%i7) M_kerucut : integrate ( dM ( z ) , z , 0 , T ) ;

\[\operatorname{ }\frac{2 \ensuremath{\pi} \rho }{3}\]

Momen inersia cakram yang massanya dM dan jejari r(z), terhadap sumbu z
dI=(1/2)(dM(z))(r(z))^2
(%i11) dI ( z ) : = ( 1 / 2 ) * dM ( z ) * r ( z ) * r ( z ) ;
I_kerucut : integrate ( dI ( z ) , z , 0 , T ) ;

\[\operatorname{ }\operatorname{dI}(z)\operatorname{:=}\frac{1}{2} \operatorname{dM}(z) \operatorname{r}(z) \operatorname{r}(z)\]

\[\operatorname{ }\frac{\ensuremath{\pi} \rho }{5}\]
