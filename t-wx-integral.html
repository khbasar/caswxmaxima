1 Dasar pengintegralan

Untuk menghitung integral, gunakan perintah integrate(f(x),x).
Menampilkan operasi integral tanpa menghitung cukup dilakukan dengan memberi tanda apostrof (') sebelum perintah integrate
--> integrate ( x ^ 2 + 3 , x ) ; ' integrate ( x ^ 2 + 3 , x ) ;

\[\operatorname{ }\frac{{{x}^{3}}}{3}+3 x\]

\[\operatorname{ }\int {\left. {{x}^{2}}+3dx\right.}\]

Integral tertentu dihitung dengan perintah integrate(f(x),x,xmin,xmax)
--> f1 ( x ) : = x ^ 2 - 3 * x + 5 $
' integrate ( f1 ( x ) , x , - 1 , 3 ) ;
integrate ( f1 ( x ) , x , - 1 , 3 ) ;

\[\operatorname{ }\int_{-1}^{3}{\left. {{x}^{2}}-3 x+5dx\right.}\]
