Ambil arah vertikal ke atas sebagai arah positif. Gaya total yang bekerja pada benda sama dengan gaya berat (arah -) ditambah gaya hambat (arah +). Persamaan gerak benda dinyatakan dengan
m*(d^2 x/dt^2)=2*m*(dx/dt)-m*g
atau
m*(dv/dt)=2*m*v-m*g
--> pers_gerak : m * ' diff ( x , t , 2 ) = - 2 * m * ' diff ( x , t ) + m * g ;
--> pers_gerak_x : m * ' diff ( x , t , 2 ) + 2 * m * ' diff ( x , t ) - m * g = 0 ;
Solusi umum persamaan gerak x(t)
--> solusi_umum_x : ode2 ( pers_gerak_x , x , t ) ;
Solusi khusus persamaan gerak x(t) dengan syarat awal x(0)=x0 dan v(0)=0
--> solusi_khusus_x : ic2 ( ( solusi_umum_x ) , t = 0 , x = x_0 , ' diff ( x , t ) = 0 ) ;
--> xt : rhs ( solusi_khusus_x ) ;
--> vt : diff ( xt , t , 1 ) ;
--> at : diff ( vt , t , 1 ) ;
--> x_0 : 0 ; g : 10 ;
Grafik x(t)
--> plot2d ( [ xt ] , [ t , 0 , 2 . 5 ] ) $
