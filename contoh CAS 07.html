Permukaan yang dinyatakan dengan fungsi z = x^2+y^2+2x-4y+10
(%i1) z : 2 · x ^ 2 + y ^ 2 + 2 · x − 4 · y + 10 ;
Plot permukaan z tersebut menggunakan plot3d
(%i2) wxplot3d ( z , [ x , − 5 , 5 ] , [ y , − 5 , 5 ] ,

Mencari titik minimum dengan pengaturan turunan sama dengan nol
(%i4) /*turunan terhadap x*/
z_x : diff ( z , x , 1 ) ;
/*turunan terhadap y*/
z_y : diff ( z , y , 1 ) ;
Titik minimum
(%i6) x_min : solve ( z_x = 0 ) ; y_min : solve ( z_y = 0 ) ;
(%i8) ttkxmin : rhs ( x_min [ 1 ] ) ; ttkymin : rhs ( y_min [ 1 ] ) ;
Nilai fungsi z di titik minimum
(%i9) define ( z ( x , y ) , z ) ; /*mendefinisikan z sebagai suatu fungsi z(x,y)*/
(%i10) float ( sublis ( [ x = ttkxmin , y = ttkymin ] , z ( x , y ) ) ) ;
