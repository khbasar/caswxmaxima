1 Sistem Koordinat Kartesian

1.1 Gradient

Operator gradien bekerja pada medan skalar. Misalkan untuk suatu medan skalar \(\phi(x,y,z)=x^2+yz\), gradiennya dinyatakan dengan
\[\vec{\nabla}\left(\phi(x,y,z)\right)=\frac{\partial \phi}{\partial x}\hat{i}+\frac{\partial \phi}{\partial y}\hat{j}+\frac{\partial \phi}{\partial z}\hat{k}\]
(%i4) /*---*/
phi ( x , y , z ) : = x ^ 2 + y * z ;
/*---*/ /*bentuk fungsi medan skalar dalam sistem koordinat kartesian*/
G : ev ( express ( grad ( phi ( x , y , z ) ) ) , diff ) $ define ( Gradien_phi ( x , y , z ) , G ) ;

\[\operatorname{ }\operatorname{phi}\left( x\operatorname{,}y\operatorname{,}z\right) \operatorname{:=}{{x}^{2}}+y z\]

\[\operatorname{ }\operatorname{Gradien\_ phi}\left( x\operatorname{,}y\operatorname{,}z\right) \operatorname{:=}\left[ 2 x\operatorname{,}z\operatorname{,}y\right] \]

1.2 Divergensi

Operator divergensi bekerja pada suatu medan vektor. Misalkan untuk suatu medan vektor
\(\vec{V1}=x^2\hat{i}+y^2\hat{j}+2z\hat{k}\)
(%i7) /*---*/
V1 ( x , y , z ) : = [ x ^ 2 , y ^ 2 , 2 * z ] ;
/*---*/ /*bentuk fungsi medan vektor dalam sistem koordinat kartesian*/
D : ev ( express ( div ( V1 ( x , y , z ) ) ) , diff ) $ define ( Divergensi_V1 ( x , y , z ) , D ) ;

\[\operatorname{ }\operatorname{V1}\left( x\operatorname{,}y\operatorname{,}z\right) \operatorname{:=}\left[ {{x}^{2}}\operatorname{,}{{y}^{2}}\operatorname{,}2 z\right] \]

\[\operatorname{ }\operatorname{Divergensi\_ V1}\left( x\operatorname{,}y\operatorname{,}z\right) \operatorname{:=}2 y+2 x+2\]

1.3 Rotasi

Operator rotasi bekerja pada suatu medan vektor. Misalkan untuk suatu medan vektor
\(\vec{V2}=x^2e^{z}\hat{i}+yz^2\hat{j}+(y+z)\hat{k}\)
(%i10) /*---*/
V2 ( x , y , z ) : = [ x ^ 2 * exp ( z ) , y * z ^ 2 , y + z ] ;
/*---*/ /*bentuk fungsi medan vektor dalam sistem koordinat kartesian*/
R : ev ( express ( curl ( V2 ( x , y , z ) ) ) , diff ) ; define ( Rotasi_V2 ( x , y , z ) , R ) ;

\[\operatorname{ }\operatorname{V2}\left( x\operatorname{,}y\operatorname{,}z\right) \operatorname{:=}\left[ {{x}^{2}} \operatorname{exp}(z)\operatorname{,}y {{z}^{2}}\operatorname{,}y+z\right] \]

\[\operatorname{(R) }\left[ 1-2 y z\operatorname{,}{{x}^{2}} {{\% e}^{z}}\operatorname{,}0\right] \]

\[\operatorname{ }\operatorname{Rotasi\_ V2}\left( x\operatorname{,}y\operatorname{,}z\right) \operatorname{:=}\left[ 1-2 y z\operatorname{,}{{x}^{2}} {{\% e}^{z}}\operatorname{,}0\right] \]

2 Sistem Koordinat Silinder

2.1 Gradient

Bentuk operator gradien dalam koordinat silinder adalah
\[\vec{\nabla}\phi(r,\theta,z)=\frac{\partial \phi}{\partial r}\hat{e_r}+\frac{1}{r}\frac{\partial \phi}{\partial \theta}\hat{e_\theta}
+\frac{\partial \phi}{\partial z}\hat{e_z}\]
(%i13) /*----*/
phicyl ( r , %theta , z ) : = r * %theta + r * z - z ^ 2 ;
/*---*/ /*bentuk fungsi medan skalar dalam sistem koordinat silinder*/
Gcyl : ev ( express ( [ diff ( phicyl ( r , %theta , z ) , r ) ,
     1 / r * diff ( phicyl ( r , %theta , z ) , %theta ) ,
   diff ( phicyl ( r , %theta , z ) , z ) ] ) , diff ) ;
define ( Gradien_phi_cyl ( r , %theta , z ) , Gcyl ) ;

\[\operatorname{ }\operatorname{phicyl}\left( r\operatorname{,}\theta \operatorname{,}z\right) \operatorname{:=}r \theta +r z-{{z}^{2}}\]

\[\operatorname{(Gcyl) }\left[ z+\theta \operatorname{,}1\operatorname{,}r-2 z\right] \]

\[\operatorname{ }\operatorname{Gradien\_ phi\_ cyl}\left( r\operatorname{,}\theta \operatorname{,}z\right) \operatorname{:=}\left[ z+\theta \operatorname{,}1\operatorname{,}r-2 z\right] \]

2.2 Divergensi

Bentuk operator divergensi dalam koordinat silinder adalah
\[\vec{\nabla}\cdot\vec{V}(r,\theta,z)=\frac{1}{r}\frac{\partial\; rV_r}{\partial r}+\frac{1}{r}\frac{\partial V_\theta}{\partial \theta}
+\frac{\partial V_z}{\partial z}\]
(%i16) /*---*/
V1cyl ( r , %theta , z ) : = [ r ^ 2 * z , 2 * cos ( %theta ) , z ] ;
/*---*/ /*bentuk medan vektor dalam sistem koordinat silinder*/
Dcyl : ev ( express ( diff ( r * V1cyl ( r , %theta , z ) [ 1 ] , r ) / r +
diff ( V1cyl ( r , %theta , z ) [ 2 ] , %theta ) / r +
diff ( V1cyl ( r , %theta , z ) [ 3 ] , z ) ) , diff ) ;
define ( Divergensi_V1cyl ( r , %theta , z ) , Dcyl ) ;

\[\operatorname{ }\operatorname{V1cyl}\left( r\operatorname{,}\theta \operatorname{,}z\right) \operatorname{:=}\left[ {{r}^{2}} z\operatorname{,}2 \cos{\left( \theta \right) }\operatorname{,}z\right] \]

\[\operatorname{(Dcyl) }3 r z-\frac{2 \sin{\left( \theta \right) }}{r}+1\]

\[\operatorname{ }\operatorname{Divergensi\_ V1cyl}\left( r\operatorname{,}\theta \operatorname{,}z\right) \operatorname{:=}3 r z-\frac{2 \sin{\left( \theta \right) }}{r}+1\]

(%i17) Divergensi_V1cyl ( 1 , %pi / 4 , 1 ) ;

\[\operatorname{ }4-\sqrt{2}\]

2.3 Rotasi

Bentuk operator rotasi dalam koordinat silinder adalah
\[\vec{\nabla}\times\vec{V}(r,\theta,z)=
\left( \frac{1}{r}\frac{\partial V_z}{\partial \theta}-\frac{\partial V_\theta}{\partial z} \right)\hat{e_r} +
\left( \frac{\partial V_r}{\partial z}-\frac{\partial V_z}{\partial r} \right) \hat{e_\theta} +
\frac{1}{r}\left( \frac{\partial (rV_\theta)}{\partial r}-\frac{\partial V_r}{\partial \theta} \right) \hat{e_z}
\]
(%i20) /*---*/
V2cyl ( r , %theta , z ) : = [ r ^ 2 * z , sin ( %theta ) , z * cos ( %theta ) ] ;
/*---*/ /*bentuk medan vektor dalam sistem koordinat silinder*/
Rcyl : ev ( [
   express ( diff ( V2cyl ( r , %theta , z ) [ 3 ] , %theta ) / r - diff ( V2cyl ( r , %theta , z ) [ 2 ] , z ) ) ,
   express ( diff ( V2cyl ( r , %theta , z ) [ 1 ] , z ) - diff ( V2cyl ( r , %theta , z ) [ 3 ] , r ) ) ,
   express ( 1 / r * diff ( r * V2cyl ( r , %theta , z ) [ 2 ] , r ) - diff ( V2cyl ( r , %theta , z ) [ 1 ] , %theta ) )
] , diff ) ;
define ( Rotasi_V2cyl ( r , %theta , z ) , Rcyl ) ;

\[\operatorname{ }\operatorname{V2cyl}\left( r\operatorname{,}\theta \operatorname{,}z\right) \operatorname{:=}\left[ {{r}^{2}} z\operatorname{,}\sin{\left( \theta \right) }\operatorname{,}z \cos{\left( \theta \right) }\right] \]

\[\operatorname{(Rcyl) }\left[ -\frac{\sin{\left( \theta \right) } z}{r}\operatorname{,}{{r}^{2}}\operatorname{,}\frac{\sin{\left( \theta \right) }}{r}\right] \]

\[\operatorname{ }\operatorname{Rotasi\_ V2cyl}\left( r\operatorname{,}\theta \operatorname{,}z\right) \operatorname{:=}\left[ -\frac{\sin{\left( \theta \right) } z}{r}\operatorname{,}{{r}^{2}}\operatorname{,}\frac{\sin{\left( \theta \right) }}{r}\right] \]

(%i21) Rotasi_V2cyl ( 2 , %pi / 2 , 0 ) ;

\[\operatorname{ }\left[ 0\operatorname{,}4\operatorname{,}\frac{1}{2}\right] \]

3 Sistem Koordinat Bola

3.1 Gradient

Bentuk operator gradien dalam koordinat bola adalah
\[\vec{\nabla}\phi(r,\theta,\varphi)=\frac{\partial \phi}{\partial r}\hat{e_r}+\frac{1}{r}\frac{\partial \phi}{\partial \theta}\hat{e_\theta}+\frac{1}{r\sin\theta}\frac{\partial \phi}{\partial \varphi}\hat{e_\varphi}\]
(%i24) /*---*/
phisph : r ^ 2 * sin ( %theta ) * sin ( %varphi ) ;
/*---*/ /*bentuk fungsi medan skalar dalam sistem koordinat bola*/
Gsph : ev ( express ( [
           diff ( phisph , r ) ,
           diff ( phisph , %theta ) / r ,
           diff ( phisph , %varphi ) / ( r * sin ( %theta ) )
       ] ) , diff ) ;
define ( Gradien_phi_sph ( r , %theta , %varphi ) , Gsph ) ;

\[\operatorname{(phisph) }\sin{\left( \theta \right) } \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } {{r}^{2}}\]

\[\operatorname{(Gsph) }\left[ 2 \sin{\left( \theta \right) } \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } r\operatorname{,}\cos{\left( \theta \right) } \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } r\operatorname{,}\cos{\left( \ensuremath{\mathrm{\% varphi}}\right) } r\right] \]

\[\operatorname{ }\operatorname{Gradien\_ phi\_ sph}\left( r\operatorname{,}\theta \operatorname{,}\ensuremath{\mathrm{\% varphi}}\right) \operatorname{:=}\left[ 2 \sin{\left( \theta \right) } \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } r\operatorname{,}\cos{\left( \theta \right) } \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } r\operatorname{,}\cos{\left( \ensuremath{\mathrm{\% varphi}}\right) } r\right] \]

(%i25) Gradien_phi_sph ( 2 , %pi / 4 , %pi ) ;

\[\operatorname{ }\left[ 0\operatorname{,}0\operatorname{,}-2\right] \]

3.2 Divergensi

Bentuk operator divergensi dalam koordinat bola adalah
\[\vec{\nabla}\cdot\vec{V}(r,\theta,\varphi)=
\frac{1}{r^2}\frac{\partial (r^2V_r)}{\partial r}+
\frac{1}{r\sin\theta}\frac{\partial(\sin\theta V_\theta)}{\partial \theta}+
\frac{1}{r\sin\theta}\frac{\partial V_\varphi}{\partial \varphi}
\]
(%i28) /*---*/
V1sph : [ r ^ 2 * sin ( %varphi ) , r * cos ( %theta ) , cos ( %varphi ) ] ;
/*---*/ /*bentuk fungsi medan vektor dalam sistem koordinat bola*/
Dsph : ev ( express ( diff ( r ^ 2 * V1sph [ 1 ] , r ) / r ^ 2 +
   diff ( sin ( %theta ) * V1sph [ 2 ] , %theta ) / ( r * sin ( %theta ) ) +
   diff ( V1sph [ 3 ] , %varphi ) / ( r * sin ( %theta ) ) ) , diff ) ;
define ( Divergensi_V1sph ( r , %theta , %varphi ) , Dsph ) ;

\[\operatorname{(V1sph) }\left[ \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } {{r}^{2}}\operatorname{,}\cos{\left( \theta \right) } r\operatorname{,}\cos{\left( \ensuremath{\mathrm{\% varphi}}\right) }\right] \]

\[\operatorname{(Dsph) }\frac{{{\cos{\left( \theta \right) }}^{2}} r-{{\sin{\left( \theta \right) }}^{2}} r}{\sin{\left( \theta \right) } r}+4 \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } r-\frac{\sin{\left( \ensuremath{\mathrm{\% varphi}}\right) }}{\sin{\left( \theta \right) } r}\]

\[\operatorname{ }\operatorname{Divergensi\_ V1sph}\left( r\operatorname{,}\theta \operatorname{,}\ensuremath{\mathrm{\% varphi}}\right) \operatorname{:=}\frac{{{\cos{\left( \theta \right) }}^{2}} r-{{\sin{\left( \theta \right) }}^{2}} r}{\sin{\left( \theta \right) } r}+4 \sin{\left( \ensuremath{\mathrm{\% varphi}}\right) } r-\frac{\sin{\left( \ensuremath{\mathrm{\% varphi}}\right) }}{\sin{\left( \theta \right) } r}\]

(%i29) Divergensi_V1sph ( 2 , %pi / 6 , 0 ) ;

\[\operatorname{ }1\]

3.3 Rotasi

Bentuk operator rotasi dalam koordinat bola adalah
\[\vec{\nabla}\times\vec{V}(r,\theta,\varphi)=
\frac{1}{r\sin\theta}\left(\frac{\partial(\sin\theta V_\varphi)}{\partial\theta} - \frac{\partial V_\theta}{\partial\varphi} \right) \hat{e_r} +
\frac{1}{r}\left( \frac{1}{\sin\theta}\frac{\partial V_r}{\partial \varphi} - \frac{\partial(rV_\varphi)}{\partial r} \right) \hat{e_\theta} +
\frac{1}{r}\left( \frac{\partial (rV_\theta)}{\partial r}-\frac{\partial V_r}{\partial\theta} \right) \hat{e_\varphi}
\]
(%i32) /*---*/
V2sph : [ r , sin ( %theta ) , cos ( %varphi ) ] ;
/*---*/ /*bentuk medan vektor dalam sistem koordinat bola*/
Rsph : ev ( express ( [
   ( diff ( sin ( %theta ) * V2sph [ 3 ] , %theta ) - diff ( V2sph [ 2 ] , %varphi ) ) / ( r * sin ( %theta ) ) ,
   ( diff ( V2sph [ 1 ] / ( sin ( %theta ) ) , %varphi ) - diff ( r * V2sph [ 3 ] , r ) ) / ( r ) ,
   ( diff ( r * V2sph [ 2 ] , r ) - diff ( V2sph [ 1 ] , %theta ) ) / r
] ) , diff ) ;
define ( Rotasi_V2sph ( r , %theta , %varphi ) , Rsph ) ;

\[\operatorname{(V2sph) }\left[ r\operatorname{,}\sin{\left( \theta \right) }\operatorname{,}\cos{\left( \ensuremath{\mathrm{\% varphi}}\right) }\right] \]

\[\operatorname{(Rsph) }\left[ \frac{\cos{\left( \theta \right) } \cos{\left( \ensuremath{\mathrm{\% varphi}}\right) }}{\sin{\left( \theta \right) } r}\operatorname{,}-\frac{\cos{\left( \ensuremath{\mathrm{\% varphi}}\right) }}{r}\operatorname{,}\frac{\sin{\left( \theta \right) }}{r}\right] \]

\[\operatorname{ }\operatorname{Rotasi\_ V2sph}\left( r\operatorname{,}\theta \operatorname{,}\ensuremath{\mathrm{\% varphi}}\right) \operatorname{:=}\left[ \frac{\cos{\left( \theta \right) } \cos{\left( \ensuremath{\mathrm{\% varphi}}\right) }}{\sin{\left( \theta \right) } r}\operatorname{,}-\frac{\cos{\left( \ensuremath{\mathrm{\% varphi}}\right) }}{r}\operatorname{,}\frac{\sin{\left( \theta \right) }}{r}\right] \]

(%i33) Rotasi_V2sph ( 2 , %pi / 2 , 0 ) ;

\[\operatorname{ }\left[ 0\operatorname{,}-\frac{1}{2}\operatorname{,}\frac{1}{2}\right] \]
