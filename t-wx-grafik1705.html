Tutorial wxMaxima: Grafik dan Gambar

1 Pilihan

Maxima menyediakan beberapa alternatif untuk menampilkan grafik atau membuat plot fungsi. Ada fungsi built-in misalnya plot2d() yang dapat langsung digunakan untuk membuat plot fungsi 2D. Untuk membuat plot 3D digunakan fungsi plot3d(). Fungsi plot2d() dan plot3d() akan menampilkan gambar di jendela (window) terpisah. Jika ingin menampilkan hasil plot langsung di jendela kerja worksheet wxmaxima, maka dapat digunakan fungsu wxplot2d() dan wxplot3d.
(%i1) wxplot2d ( x ^ 2 , [ x , - 2 , 2 ] ) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

(%i2) wxplot3d ( - ( x * y ) ^ 2 , [ x , - 2 , 2 ] , [ y , - 2 , 2 ] ) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

Alternatif lain untuk membuat gambar atau grafik adalah dengan terlebih dahulu memanggil paket khusus yaitu paket "draw". Dengan menggunakan paket "draw" ini, pembuatan plot dan grafik dapat lebih kaya fitur dan pengaturannya lebih mudah. Untuk menggunakan paket "draw", terlebih dahulu harus dipanggil menggunakan perintah load("draw"). Selanjutnya fungsi yang digunakan untuk menampilkan grafik atau membuat plot adalah draw2d(), draw3d() dan seperti halnya keterangan di atas, bila ingin menampilkan grafik langsung pada jendela kerja sebagai output, maka fungsi yang digunakan adalah wxdraw2d() dan wxdraw3d()
(%i3) load ( "draw" ) ;

2 Grafik dan Plotting 2D

2.1 Plot fungsi eksplisit

Untuk menampilkan grafik fungsi eksplisit, secara sederhana dapat dilakukan dengan wxdraw2d()
(%i4) wxdraw2d ( explicit ( 2 * x ^ 2 - 5 * sin ( x ) , x , - 5 , 5 ) ) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

(%i6) f ( x ) : = 3 * x ^ 3 - 5 * x ^ 2 $ wxdraw2d ( explicit ( f ( x ) , x , - 2 , 2 ) ) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

Menggambarkan beberapa fungsi dalam satu grafik
(%i9) f1 ( x ) : = 2 - 5 * x ^ 2 $ f2 ( x ) : = 3 * x ^ 3 - 5 * x ^ 2 $
wxdraw2d (
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   explicit ( f2 ( x ) , x , - 2 , 2 )
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

2.2 Plot fungsi implisit

Sedangkan untuk membuat plot fungsi implisit, misalnya berbentuk persamaan lingkaran \(x^2+y^2=1\), dapat dilakukan dengan perintah implicit()
(%i11) f3 : x ^ 2 + y ^ 2 = 1 $
wxdraw2d ( implicit ( f3 , x , - 2 , 2 , y , - 2 , 2 ) ) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

Gambar di atas, yang seharusnya menunjukkan suatu bentuk lingkaran, terlihat tidak proporsional, namun hal itu disebabkan pengaturan nilai sumbu-sumbu.

3 Pengaturan Tampilan Grafik

3.1 Penambahan Legenda

Penambahan legenda dapat dilakukan dengan opsi "key-color"
(%i12) wxdraw2d (
   key = "fungsi pertama" , color = blue ,
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   key = "fungsi kedua" , color = red ,
   explicit ( f2 ( x ) , x , - 2 , 2 )
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

3.2 Nama sumbu (label) dan judul (title)

Label sumbu (axis label) diatur dengan option xlabel dan ylabel, sedangkan judul grafik diatur dengan option title
(%i13) wxdraw2d (
   key = "fungsi pertama" , color = blue ,
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   key = "fungsi kedua" , color = red ,
   explicit ( f2 ( x ) , x , - 2 , 2 ) ,
   xlabel = "variabel x" , ylabel = "variabel y" ,
   title = "Plot 2 fungsi"
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

3.3 Membatasi area grafik

Mengatur dan membatasi area grafik dengan option xrange dan y range
(%i14) wxdraw2d (
   key = "fungsi pertama" , color = blue ,
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   key = "fungsi kedua" , color = red ,
   explicit ( f2 ( x ) , x , - 2 , 2 ) ,
   xlabel = "variabel x" , ylabel = "variabel y" ,
   title = "Plot 2 fungsi" ,
   xrange = [ - 3 , 3 ] , yrange = [ - 25 , 10 ]
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

3.4 Pengaturan tebal garis

Tebal garis atau kurva diatur dengan option line_width
(%i15) wxdraw2d (
   key = "fungsi pertama" , color = blue ,
   line_width = 2 ,
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   key = "fungsi kedua" , color = red ,
   line_width = 4 ,
   explicit ( f2 ( x ) , x , - 2 , 2 ) ,
   xlabel = "variabel x" , ylabel = "variabel y" ,
   title = "Plot 2 fungsi" ,
   xrange = [ - 3 , 3 ] , yrange = [ - 25 , 10 ]
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

3.5 Menampilkan sumbu-sumbu koordinat

Untuk menampilkan sumbu-sumbu koordinat dalam bentuk konvensional, dapat dilakukan dengan option xaxis=true dan yaxis=true
Pengaturan tipe garis sumbu dengan option xaxis_type dan yaxis_type
(%i16) wxdraw2d (
   key = "fungsi pertama" , color = blue ,
   line_width = 2 ,
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   key = "fungsi kedua" , color = red ,
   line_width = 4 ,
   explicit ( f2 ( x ) , x , - 2 , 2 ) ,
   xlabel = "variabel x" , ylabel = "variabel y" ,
   title = "Plot 2 fungsi" ,
   xrange = [ - 3 , 3 ] , yrange = [ - 25 , 10 ] ,
   xaxis = true , yaxis = true ,
   xaxis_type = solid , yaxis_type = solid
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

3.7 Fungsi potongan (piecewise function)

Yang dimaksud fungsi potongan adalah fungsi yang didefinisikan dalam interval-interval tertentu. Misalnya
\[
f(x)=\begin{cases} x+1,&\;-1<x<0 \\ 1, & 0<x<1 \end{cases}
\]
(%i20) wxdraw2d (
   title = "Fungsi Potongan" ,
   yrange = [ - 2 , 2 ] ,
   line_width = 4 ,
   xaxis = true , yaxis = true , xaxis_type = solid , yaxis_type = solid ,
   explicit ( x + 1 , x , - 1 , 0 ) ,
   explicit ( 1 , x , 0 , 1 )
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

3.8 Menambahkan label dan titik pada grafik

Label berupa teks atau keterangan tulisan lain dapat diberikan dengan option label().
Formatnya adalah
label(["teks label",x,y])
dengan x dan y menyatakan posisi label
Teks label tersebut juga diatur alignment nya dengan perintah label_alignment
(%i24) wxdraw2d (
   key = "fungsi pertama" , color = blue ,
   line_width = 2 ,
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   key = "fungsi kedua" , color = red ,
   line_width = 4 ,
   explicit ( f2 ( x ) , x , - 2 , 2 ) ,
   xlabel = "variabel x" , ylabel = "variabel y" ,
   title = "Plot 2 fungsi" ,
   xrange = [ - 3 , 3 ] , yrange = [ - 25 , 10 ] ,
   color = black , label ( [ "label tambahan 1" , - 1 , 3 ] ) ,
   color = purple , label ( [ "label lainnya ??" , 1 , - 3 ] )
) ;

\[\operatorname{ }\]


\[\operatorname{ }\]

Sedangkan untuk menambhakan titik dilakukan dengan perintah
points([[x1,y1],[x2,y2],...,[xn,yn]])
Tipe titik diatur dengan point_type
(%i29) wxdraw2d (
   key = "fungsi pertama" , color = blue ,
   line_width = 2 ,
   explicit ( f1 ( x ) , x , - 2 , 2 ) ,
   key = "fungsi kedua" , color = red ,
   line_width = 4 ,
   explicit ( f2 ( x ) , x , - 2 , 2 ) ,
   xlabel = "variabel x" , ylabel = "variabel y" ,
   title = "Plot 2 fungsi" ,
   xrange = [ - 3 , 3 ] , yrange = [ - 25 , 10 ] ,
   color = black , label ( [ "label tambahan 1" , - 1 , 3 ] ) ,
   color = purple , label ( [ "label lainnya ??" , 1 , - 3 ] ) ,
   color = black , point_type = filled_circle , point_size = 5 , key = "titik tambahan" ,
   points ( [ [ 0 , 0 ] , [ 1 , 0 ] ] )
) ;
